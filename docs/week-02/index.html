Week 2 — Context: The Smart Zone

The context window is the resource everything else depends on. This week you learn why output degrades as it fills, how to reset without losing what matters, how to give every session durable memory with a lean CLAUDE.md, and how to steer with plan mode and rewind — then you prove the difference experimentally.

Estimated total time: 3.5–4.5 hours (readings/videos + quiz + assignment + meeting). These pages cover the same ground as the assigned readings — use them as your primary lesson text, and the linked sources to go deeper.

This week's learning pages

Six mastery objectives, seven lessons, in recommended order:

  • M07Understand
    The Smart Zone and the Dumb Zone

    Why context occupancy degrades output — attention degradation — and how to judge when a session should be reset.

  • M08Apply
    The Memento Principle: /clear over /compact

    Externalize state to files and start empty, rather than trusting a lossy summary — and the honest tradeoff between the two.

  • M09Create
    Writing a Lean CLAUDE.md

    What earns a line, the pruning test, and justifying what you deliberately left out — the A2 artifact.

  • M10Understand
    CLAUDE.md Scopes — and When to Use a Skill

    User, project, local, and path-scoped instruction files; how they load; and when content belongs in a skill or hook instead.

  • M11 · 1 of 2Apply
    Plan Mode: Scope Before You Execute

    Read-and-propose mode: agree on the approach while a veto still costs nothing, then execute on the context planning built.

  • M11 · 2 of 2Apply
    Rewind: Course-Correct Without Starting Over

    Checkpoints, the rewind menu's restore and summarize options, and the limitations that keep git in your workflow.

  • M12Evaluate
    The Session Experiment: Fresh vs. Polluted

    Run the same task in both conditions, measure with a pre-committed rubric, and write a comparison a skeptic could check.

Suggested path

Concepts first (M07 → M08), then the CLAUDE.md pair — skim M10 for the map, write yours with M09 — then the steering pair (M11) hands-on in one sitting. Do M12 last, immediately before writing your A2 comparison: it uses the CLAUDE.md you just wrote and every habit from the week.

How this week is assessed

  • Quiz 2 (25 MCQs): context occupancy and output quality; CLAUDE.md scope hierarchy; /clear vs. /compact tradeoffs; plan mode and rewind. Every lesson's self-check is written in the quiz's application-over-recall style.
  • Assignment A2 (10 pts): write a lean CLAUDE.md (under 100 lines) for your Week 1 repository and note, in comments, two things you deliberately excluded and why. Then run the same moderate task twice — once in a fresh, well-contexted session and once at the end of a long polluted session — and write a half-page comparison of the results.
