Agentic Coding with Claude Week 1 hub

Week 1 — Orientation: What Is AI Labor?

What an agentic coding tool actually is, how it differs from the AI assistance you already know, and your first real session — installed, initialized, committed, and reflected on.

Estimated total time: 3.5–4.5 hours (readings/videos + quiz + assignment + meeting). These pages cover the same ground as the assigned readings — use them as your primary lesson text, and the linked sources to go deeper.

This week's learning pages

Six mastery objectives, eight lessons, in recommended order:

  • M01 · 1 of 2Understand
    The Agentic Loop

    Gather context → take action → verify → repeat, and why model + harness = agent.

  • M01 · 2 of 2Understand
    Tools & Permissions

    The five tool categories, what Claude can access, and the permission modes that keep you in control.

  • M02Analyze
    What Is AI Labor?

    Agentic coding vs. chat vs. autocomplete, and the measured division of planning and execution decisions.

  • M03 · 1 of 2Apply
    Install & Authenticate

    The one-line installer for your OS, and signing in with your Pro account.

  • M04Apply
    Core Session Controls

    Prompting, interrupting, /help, /clear, and reviewing every diff before you commit.

  • M05Analyze
    Whose Decision Was It?

  • M06Apply
    GitHub Basics: Repo, Clone, Commit

Suggested path

Concepts first (M01 → M02), then hands-on in one sitting: create your repo (M06), install and initialize (M03), and exercise the session controls (M04). Finish with M05 right before writing your A1 reflection — it's the template for it.

How this week is assessed

  • Quiz 1 (25 MCQs): the agentic loop; tools and permissions; what distinguishes agentic coding from chat assistance; findings on the human/agent division of decisions. Every lesson's self-check is written in the quiz's application-over-recall style.
  • Assignment A1 (10 pts): install Claude Code and authenticate with your Pro account; create a GitHub repository; run /init; complete one small end-to-end task (e.g., a working CLI utility) with a clean commit; submit the repo link plus a five-sentence reflection identifying which decisions were yours and which were Claude's.
